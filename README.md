# catch-me-if-you-can
A simulation teting whether an AI agent can pursue a hidden goal while deceiving a safety monitor in plain language scenarios 
# overview
This project explores a core question in AI safety: can an AI agent complete a hidden objective while convincing a monitor agent it is behaving normally? Built as part of research aligned with Anthropic's AI control and adversarial robustness priorities.
